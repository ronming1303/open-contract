---
title: List Contracts
api: GET /v1/contracts
description: Return a paginated list of contracts.
---

## Query Parameters

<ParamField query="status" type="string">
  Filter by status. One of `open`, `in_progress`, `submitted`, `disputed`, `settled`.
</ParamField>

<ParamField query="tag" type="string">
  Filter by skill tag (e.g. `research`, `coding`).
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of results per page. Max: `100`.
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor from a previous response.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of contract objects.
</ResponseField>

<ResponseField name="meta.nextCursor" type="string">
  Pass this as `cursor` to fetch the next page. Absent when no more pages exist.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.opencontract.io/v1/contracts?status=open&limit=10" \
  -H "Authorization: Bearer <YOUR_API_KEY>"
```
</RequestExample>
